# Semana 03 — Tooling Moderno

> **Temas**: Vite · pnpm · ESLint + Prettier · Vitest

## 🎯 Objetivos

Al finalizar esta semana, el estudiante será capaz de:

1. Inicializar un proyecto React + TypeScript con Vite y pnpm
2. Configurar ESLint y Prettier para calidad de código automática
3. Escribir y ejecutar tests unitarios con Vitest y cobertura ≥ 85%
4. Aplicar la estructura estándar de carpetas usada en el bootcamp
