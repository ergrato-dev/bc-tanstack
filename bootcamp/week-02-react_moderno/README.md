# Semana 02 — React Moderno

> **Etapa 0 · Semana 2 de 3** | Prerequisito: [Semana 01 — TypeScript Esencial](../week-01-typescript_esencial/README.md)

## 🎯 Objetivos

Al finalizar esta semana, el estudiante será capaz de:

1. Gestionar estado local y efectos con `useState`, `useEffect` y `useRef`
2. Optimizar renders con `useMemo` y `useCallback`
3. Componer componentes con separación de responsabilidades y manejar errores con Error Boundaries
4. Compartir estado global sin librerías externas usando Context API
